# Swisstronik Tesnet Techinal Task 6 (Deploy Proxy)

## use Gitpod or directly from your local computer (linux/macOS) only


### 1. Clone Repository

```bash
git clone https://github.com/alkindivv/proxy-swisstronik.git
```

```bash
cd proxy-swisstronik
```

### 2. Install Dependency

```bash
npm install
```

### 3. Set .env File

create .env file in root project

```bash
touch .env
```

add this to your .env file

```bash
PRIVATE_KEY="your private key"
```

### 4. Compile Smart Contract

```bash
npm run compile
```

### 5. Deploy Smart Contract

```bash
npm run deploy
```

### 6. Initialize Owner

```bash
npm run initialize
```

### 7. Add Issuer

```bash
npm run add-issuers
```

### 8. Get Issuers list

```bash
npm run list-issuers
```

### 9. Upgrade Smart Contract

```bash
npm run upgrade
```

### 10. Finsihed
